## Фаза 3 — тяжёлый DSP (всё в браузере, сервер не трогаем)

Строится поверх аудио-графа в [web/src/lib/audio.ts](web/src/lib/audio.ts) (класс `LocalAudio`,
цепочка `source → gain → gate → mute → dest`). Вставлять узлы между `source` и `gain`/`gate`.

- [ ] **RNNoise** (нейрошумодав): WASM + AudioWorklet, вставить как узел в граф; вариант шумодава
      «выкл / браузерный / RNNoise» в панель [Settings.tsx](web/src/components/Settings.tsx). Сделать opt-in
      (может грузить слабые телефоны). Библиотека: `@jitsi/rnnoise-wasm` или аналог; worklet на 48кГц/480.
- [ ] **Эквалайзер** — несколько `BiquadFilterNode` (low/mid/high) + ползунки.
- [ ] **Компрессор** — `DynamicsCompressorNode` + пресеты/ползунки.
- [ ] **Фан-фильтры / «радио»** — пресеты (bandpass + waveshaper-дисторшн), выбор в панели.
- [ ] **Локальная запись** — `MediaRecorder` на клиенте (микс: локальный + удалённые через
      `AudioContext`), скачивание файла. Сервер ничего не пишет. Уведомление о записи — НЕ нужно.
